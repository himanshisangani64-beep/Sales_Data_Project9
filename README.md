# Sales Data Analyzer & Visualization

## Description

Sales Data Analyzer & Visualization is a Python-based  application developed to perform data analysis, cleaning, statistical operations, and visualization on CSV sales datasets. The project uses Pandas for data manipulation, NumPy for numerical operations, and Matplotlib and Seaborn for creating various charts. It provides an interactive menu-driven interface that allows users to explore datasets, handle missing values, perform analytical operations, generate descriptive statistics, and visualize data effectively.

---

## Objective

The main objectives of this project are:

- Analyze sales datasets using Python.
- Perform data cleaning and preprocessing.
- Apply statistical and numerical operations on data.
- Generate descriptive statistics and pivot tables.
- Create multiple types of data visualizations.
- Save generated visualizations as image files.

---

## Assumptions

- The input dataset must be in CSV format.
- The dataset should contain valid column names.
- If available, the `OrderDate` column should follow the `DD-MM-YYYY` format.
- Numerical operations require numeric columns.
- Pivot tables require valid index, value, and aggregation columns.
- Visualizations require appropriate categorical and numerical columns.
